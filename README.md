# How to run benchmark

1. Run `pip install -r requirements.txt`
2. Run `data_server.py`, `data_repeater1.py`, `data_repeater2.py` in both `perf_demo_gRPC/` and `perf_demo_REST/`
3. Run `benchmark.py` in `perf_demo_benchmark`
